# AERO Frontend

This directory contains the frontend source code for the AERO platform. It is designed to be a modern, responsive, and high-fidelity React application.

## Folder Structure

- **src/**: The main source directory for all React components and logic.
- **public/**: Static assets like `index.html`, `favicon.ico`, and manifest files.
- **components/**: Reusable UI components used across multiple pages.
- **pages/**: Individual page components that correspond to application routes.
- **hooks/**: Custom React hooks for shared logic and state management.
- **services/**: Services for handling API calls and external data fetching.
- **utils/**: Shared utility functions and helper methods.
- **assets/**: Media files such as images, icons, and global CSS/SCSS files.
