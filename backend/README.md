# AERO Backend

This directory contains the backend API source code for the AERO platform, powered by Node.js and Express.

## Folder Structure

- **src/**: Core server logic and entry point for the backend application.
- **controllers/**: Functions that handle incoming requests and return responses.
- **routes/**: Definition of API endpoints and mapping to controllers.
- **models/**: Data models and schemas for database interactions (e.g., Mongoose models).
- **middleware/**: Custom middleware for authentication, logging, and error handling.
- **config/**: Configuration files for environment variables and database connections.
- **utils/**: Helper functions and utility modules for backend operations.
